# specs/

Source-of-truth OpenAPI specs consumed by `@archastro/sdk-generator` to
emit typed SDKs and contract tests.

## `platform-openapi.json`

The ArchAstro Platform API surface — REST endpoints plus Phoenix
`x-channels`. Generated by the backend monorepo's API DSL registry:

```bash
mix api_dsl.gen.docs.openapi \
  --registry ArchAstro.PlatformWeb.ApiRegistry \
  --out platform-openapi.json
```

This file is committed so downstream SDK repos (e.g. `archastro-js`,
`archastro-python`) can pull a stable spec from a known URL without
depending on a private backend checkout. Update it from the backend
whenever the API surface changes; SDKs re-run their `regenerate_sdk.sh`
to pick up the new version.
